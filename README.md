# Marvel Frontend

### Clone the Project

To start, clone the project using the following command:

### If you using _yarn_ make this command line :

To install all dependences :

```bash
yarn
```

To launch your development sever :

```bash
yarn dev
```

### If you using _npm_ make this command line :

To install all dependences :

```bash
npm install
```

To launch your development sever :

```bash
npm run dev
```
